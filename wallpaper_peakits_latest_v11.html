About

Lumina Aeris

This web app generates a lock-screen wallpaper based on your location, current weather, time of day, moon phase, and a rotating seasonal theme. It also caches points of interest (POIs) per location, includes a prompt builder, and supports multiple image models and quality levels. (Replace this text later with your final About content.)

Title animation & weather FX

On page load, the app name does a lightweight animated color-and-light shimmer. After weather is fetched (when you generate a wallpaper), subtle weather-themed effects may appear behind the title (clear / clouds / rain / snow / thunder).

Image dimensions

Default output dimensions use this device’s current screen size. You can override it with presets like iPhone 17 Pro, iPhone 17 Pro Max, iPad, and Desktop 16:9.

Advanced

Cloudflare Worker profile

Export your current settings into a Worker-hosted profile so iOS Shortcuts can call a single URL that returns an image (no UI). Requirements: a Cloudflare Worker endpoint and a KV namespace bound to that Worker.

Worker secrets (required)

The Pollinations API key is not stored in the generated profile by default. You must set it as a Cloudflare Worker secret named POLLINATIONS_API_KEY.

POLLINATIONS_API_KEY = <YOUR_POLLINATIONS_API_KEY> ADMIN_TOKEN = <YOUR_ADMIN_TOKEN> (optional, protects /profile)

If you use ADMIN_TOKEN, paste it below (or use it only in Cloudflare and leave this blank).

Example: https://wallpaper.peakits.com (no trailing slash).

This becomes the KV key (example: default).

Only used to save/update profiles. Not used for generating images.

Security tip: prefer Worker secrets; avoid placing API keys in URLs or shared profile JSON.

Worker setup (guide)

1) Create a Worker (Workers & Pages → Create). 2) Create/bind a KV namespace named PROFILE_KV. 3) Add secrets: POLLINATIONS_API_KEY (required), ADMIN_TOKEN (recommended). 4) Add routes: /profile (POST) to save profiles, /wallpaper (GET) to generate an image.

Limits reminder: Workers Free 100,000 requests/day; KV Free 100,000 reads/day and 1,000 writes/day.
